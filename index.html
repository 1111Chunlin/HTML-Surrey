Abstract

Active Learning (Angluin, 1988; Cohn et al., 1994) is a paradigm where a concept is learnt by an
intelligent machine through interaction with an oracle across learning epochs. A distinctive
component is the querying strategy (e.g., Query by Committee, Monte Carlo estimation, uncertainty
sampling, margin sampling). Newly labeled instances are added to the training set and the model
is updated iteratively until a stopping criterion is met (e.g., as in Vlachos, 2008).
